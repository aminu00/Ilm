# عِلم — Ilm | Islamic Q&A Platform

Ilm is a premium, modern Islamic Q&A platform designed to connect seekers of knowledge with verified scholars. Built with a focus on authenticity, ease of use, and multi-lingual support.

## 🌟 Features

- **Authentic Answers**: Only verified scholars can provide answers to questions.
- **Multilingual Support**: Supports English, Arabic, Yoruba, Hausa, Igbo, and Pidgin.
- **Community Engagement**: Join communities, share posts, and discuss topics with fellow Muslims.
- **Anonymous Asking**: Ask sensitive questions anonymously.
- **Scholar Profiles**: Follow your favorite scholars and stay updated with their latest answers.

## 🚀 Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Backend/Auth**: Supabase (PostgreSQL)
- **Mobile**: Capacitor (iOS/Android ready)

## 🛠️ Setup

1. **Clone the repository**
2. **Install dependencies**: `npm install`
3. **Configure Environment**: Create a `.env` file with your Supabase credentials.
4. **Run Development Server**: `npm run dev`

## 📖 Database Migration
All mission-critical SQL migrations are located in `supabase/migrations/`. These should be applied to your Supabase project using the SQL Editor.
